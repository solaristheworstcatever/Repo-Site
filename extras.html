Level editors

Recommended level editors:

Henry's Leditor

Has a good UI and is easy to use, Its fairly similar to Rained, but not as frequently updated.

Rained

Has a good UI and is easy to use, the creator is always open to suggestions.

Ogscule Editor

A smoother and better version of Rain World Editor Plus.

Other level editors:

Community Editor

Combination of the Drought, MSC, Downpour, Official and Substratum editors. Allows to add custom effects easier than other editors do.

RWE+

Timofey's first custom level editor. While Timofey is making a second one, this one is not recommended to use.

Monsoon

Was made by MACMAN2003 alongside RWE+, but it has been abandoned.

Official

The original editor, released in 2018. It is here for archivial purposes, and its use is discouraged.

Downpour

Released in 2023, It is here for archivial purposes, and its use is discouraged.

Useful resources

Main Wiki

The official Rain World wiki, containing information about all items in the game.

Modding Wiki

A place to share and obtain information about modding Rain World. Feel free to contribute!

Mods Wiki

A place to collect information on content mods for Rain World. Feel free to contribute!

Region Lease

A spreadsheet used to help ensure region mods don't conflict with each other.

Base Game Interactive Map

Detailed maps of all base game and MSC regions.

Mod Interactive Map

Detailed maps of several modded regions.

Tools

Lease Checker

A tool to easily check which region IDs are claimed on the lease.

Tile Init Editor

A tool that gives a simple and easy-to-use UI for creating Level Editor tile init lines.

Prop Renderer

A scene and rendering workflow for Blender that slices and converts closed meshes into prop sprite sheets.

Drizzle

Faster renderer that comes with, and is used by most level editors.

FloodForge

An in-develpment tool used to manipulate world files and create in-game maps for regions.

World Editor

An older tool used to manipulate world files and create in-game maps for regions.

Cornifer

A tool used to make wiki style maps.

Room Prompt Generator

Tool for generating random room prompts for when you lack inspiration.

Andrew Editor

A tools that allows for limited editing of the pre-existing Rain World levels.

Adobe Director

The software used to make the Official, Downpour, and Comms Editors. It can be used to edit the Comms code to add new effects.
